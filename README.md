# gesco-frontend


## Prerequire
Populate first your .env.local. You can use the example at .env.exemple
## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

## Docker setup
```
docker-compose build
docker-compose up
```
